Comparación de Librerías de Gráficos

Tres implementaciones del mismo chart de sesiones para evaluar pros/cons

Recomendación
Para Scout Dashboard, Recharts es la mejor opción: código declarativo React, bundle razonable (~90KB), interactividad built-in, y muy fácil de mantener. Chart.js es excelente si necesitas vanilla JS, pero en un stack React puro es más código. Canvas nativo es solo para casos edge donde bundle size es crítico (~5KB total).
